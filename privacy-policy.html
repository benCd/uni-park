This privacy policy will specify which and how user data is stored and handled.

1 User Data Stored

1.1 Private (Non Public) Data

  • User Password
  • User Email Address
  • User Cookie
  • User Identifying Data
  • User linked GPS data
  • Event IDs for Google Calendar events

1.2 Public Data

  • Generalised GPS Data aggregated from user linked GPS Data
  • Time series GPS Data

2 User Information Handling

All data will be sent using secure connections. You can ensure that you are safely connected in the browser by checking for the green lock and "https" in the address bar. We also ensure that data transferred between our application and our server is encrypted.

2.1 Private User Information

Private data will be used for analysis purposes only. We will never sell or share data that can be linked to you in any way with a third party.

Every report you make to the project will be deleted after one month when it has been processed for statistical and analytic purposes.

2.2 Public User Information

Certain information like the product of operations on private user data will be publicly available. These products will be anonymised and cannot be linked to any user. By signing up for Uni Park you volunteer your private data for the production of this publicy available data.

3 Access to Information

You will at any point be able to see all the information we keep on you (except for your password and email address) from the app. This way we assure that you are actually logged in. Nobody besides you and certain staff will be able to access your private data. We will only access your data if we need to use it for preparation or to contact you. We will never send anyone your password or make it available.
